래퍼 클래스  ( 객채 자료형): 
int(기본 자료형) 비슷한 객체 자료형 =  Integer

자료 구조: 여러가지 값을 저장하는 공간
        배열
        Stack:
        Queue:
        컬랙션 프레임 워크: 자바에서 여러가지 값을 저장
            ArrayList
            HashSet
            HashMap

List :  인터페이스 = 집합을 구현한 자료구조
        자식클래스  = ArrayList
                    LinkedList (중간 삭제시 속도빠름)
        특징 =   순번(인텍스 번호) 
                중복된 값 저장 가능

Set:    인터페이스 
        자식 클래스 :  HashSet
        특징 =    순번 없음
                 중복된 값 저장 안됨

Map: 인터페이스 (키, 값) 1쌍을 저장하는 자료구조
        자식클래스 =   HashMap
        특징 =   키는 중복된 값 저장안된, 
                값은 중복된 값 저장 가능
                순번 없음

Stack: LIFO (First in last out)  자료구조
Queue: FIFO (first in first out) 자료구조
